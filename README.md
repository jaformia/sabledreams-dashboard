# SableDreams Dashboard

Starter dashboard scaffold built with Next.js App Router.

## Setup

1. Install dependencies:
   `npm install`
2. Create your local environment file:
   `Copy-Item .env.example .env`
3. Start development:
   `npm run dev`

## Included structure

- Auth pages: login, register, forgot password
- Dashboard pages: overview, analytics, users, products, settings
- Shared UI components, layout components, placeholder charts
- Basic hooks, services, store, types, constants, and middleware

## GitHub

- `.env` is ignored by Git
- `.env.example` is safe to commit
- Initialize a repository and connect a remote:
  `git init`
  `git remote add origin <your-github-repo-url>`
